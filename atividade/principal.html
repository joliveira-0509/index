COMANDOS PRINCIPAIS

git config: Configura as informações do usuário, como nome e email.

1. git config --global.name "Seu nome"

2. git config --global.user "seuemail@exemplo.com"

git init: inicializa um novo repositório Git

1. git init

git clone: clona um repositório Git

1. git clone "URL do repositório"

git add: Adiciona arquivos ao índice (staging area).

1. git add *arquivo*

2. git add .

git commit: Confirma as mudanças no repositório

1. git commit -m "Mensagem do commit"

Comandos de Ramificação e Mesclagem

1. git branch: lista, cria ou deleta branches

git brach git brach *nome-do-branch*

2. git checkout: muda para outro branch.

git checkout *nome-do-branch*

3. git merge: Mescla um branch com o branch atual.

git merge *nome-do-branch*

Comandos de sincronização

1. git pull: Atualiza o repositório local com mudanças do repositório remoto.

git pull

2. git push: Envia as mudanças locais para o repositório remoto.

git push
